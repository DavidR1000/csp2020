My Imaginary World




In this project, we coded it so a picture would be drawn using turtle methods. In the picture the first thing that would be drawn is the background which includes the grass, sky, sun, and clouds. Next, the house would be drawn using colors that the user want by using the input() method. Then, the garden would drawn which includes 2 flowers, a ladybug with the wing color decided by the use, and a spider with the color decided by the user. The program would end off with having you chose between a plane, bee, and hot air balloon and depending on which the user choses is the object that will move across the screen.
